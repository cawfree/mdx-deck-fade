# mdx-deck-fade
A &lt;Fade/> component for mdx-deck.

## 🚀 Getting Started

Using [`npm`]():

```sh
npm install --save mdx-deck-fade
```

Using [`yarn`]():

```sh
yarn add mdx-deck-fade
```

## ✍️ Example

```javascript
import Fade from 'mdx-deck-fade';

<Fade
>
  Press right to fade in!
</Fade>

```
